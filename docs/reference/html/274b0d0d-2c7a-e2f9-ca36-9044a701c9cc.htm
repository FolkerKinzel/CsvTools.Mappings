      Parameters

      filePath  String
      File path of the CSV file to read.
      mapping  CsvRecordMapping
      The CsvRecordMapping used to convert the CSV data.
      conversion  FuncObject, TResult

      A function that converts the content of mapping to an instance of TResult.

      The function is called for each row in the CSV file and gets the CsvRecordMapping as argument, filled with the current CsvRecord instance. The CsvRecordMapping is passed to the function as dynamic argument: Inside the function the registered DynamicProperty instances can be used like regular .NET properties, but without IntelliSense ("late binding").

      isHeaderPresent  Boolean  (Optional)
      true, to interpret the first line as a header, otherwise false.
      options  CsvOpts  (Optional)
      Options for reading the CSV file.
      delimiter  Char  (Optional)
      The field separator character.
      textEncoding  Encoding  (Optional)
      The text encoding to be used to read the CSV file or null for UTF8.
