      MappingExtension Methods

      Methods

      AddPropertyT(Mapping, String, MultiColumnTypeConverterT) Adds a new DynamicProperty instance to the Mapping, which accesses several columns of a CSV file.
      AddPropertyT(Mapping, String, TypeConverterT) Adds a new DynamicProperty instance to the Mapping, which accesses a single column of a CSV file with header with the column name.
      AddPropertyT(Mapping, String, IEnumerableString, TypeConverterT) Adds a new DynamicProperty instance to the Mapping, which accesses a single column of a CSV file with header with a collection of column name aliases.
      AddPropertyT(Mapping, String, Int32, TypeConverterT) Adds a new DynamicProperty instance to the Mapping, which accesses a single column of the CSV file with its index.
      Clear Removes all DynamicProperty instances from the Mapping.
