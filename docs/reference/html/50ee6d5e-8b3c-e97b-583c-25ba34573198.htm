      Parameters

      filePath  String
      File path of the CSV file.
      analyzedLines  Int32  (Optional)
      Maximum number of lines to analyze in the CSV file. The minimum value is AnalyzedLinesMinCount. If the file has fewer lines than analyzedLines, it will be analyzed completely. (You can specify Int32.MaxValue to analyze the entire file in any case.)

      Return Value

      ValueTupleEncoding, CsvAnalyzerResult
      The results of the analysis.

      Remarks

      The method performs a statistical analysis on the CSV file to find the appropriate parameters for reading the file. The result of the analysis is therefore always only an estimate, the accuracy of which increases with the number of lines analyzed.

      The analysis is time-consuming because the CSV file has to be accessed for reading.

      This method also automatically determines the Encoding of the CSV file.
