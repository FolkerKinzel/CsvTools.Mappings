      Remarks

      A ready-to-use implementation of this class can't be provided because their structure depends on the CSV file to be processed. Fortunately, writing a derived class is easy:

        Notes to Implementers

      Pass a CsvMapping instance that targets the required columns of the CSV file to the constructor, and override the abstract members. (For overriding Clone using the copy constructor (MultiColumnTypeConverterT(MultiColumnTypeConverterT)) is required!)
