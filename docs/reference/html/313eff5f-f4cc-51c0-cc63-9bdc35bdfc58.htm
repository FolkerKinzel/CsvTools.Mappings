      mappingBuilder  CsvRecordMappingBuilder

      A CsvRecordMappingBuilder that builds the CsvRecordMapping to use to access those columns of the CSV file that are required for the Type conversion.

      Add all required properties to mappingBuilder before passing it as argument. (The Build method is called from the constructor.)

      defaultValue  T
      The DefaultValue to return when a parsing error occurs and the Throwing property is false.
      throwing  Boolean
      Sets the value of the Throwing property.
