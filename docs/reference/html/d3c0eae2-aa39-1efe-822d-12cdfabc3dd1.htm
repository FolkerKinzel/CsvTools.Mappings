      resetExcess  Boolean  (Optional)
      If data has fewer items than mapping has DynamicProperty instances and this parameter is true, the surplus properties in record will be reset to their DefaultValue. For performance reasons this parameter can be set to false when writing CSV because WriteRecord resets all fields in mapping.

      Usage Note

      In Visual Basic and C#, you can call this method as an instance method on any object of type Mapping. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).

      Exceptions

      ArgumentNullExceptionmapping or data is null.
      ArgumentOutOfRangeExceptiondata has more items than the number of DynamicProperty instances in mapping.
      InvalidOperationException The Record property of mapping is null. Assign a CsvRecord instance to mapping before calling this method.
      InvalidCastException An item in data does not match the expected data type.
      FormatException One of the TypeConverterT instances uses an invalid format string.
