      csv  String

      mapping  Mapping
      The Mapping used to convert the CSV data.
      conversion  FuncObject, TResult

      A function that converts the content of mapping to an instance of TResult.

      The function is called for each row in csv and gets the Mapping as argument, filled with the current CsvRecord instance. The Mapping is passed to the function as dynamic argument: Inside the function the registered DynamicProperty instances can be used like regular .NET properties, but without IntelliSense ("late binding").
