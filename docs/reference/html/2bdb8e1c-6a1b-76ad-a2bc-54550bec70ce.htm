      mappingBuilder  CsvMappingBuilder

      A CsvMappingBuilder that builds the CsvMapping to use to access those columns of the CSV file that are required for the Type conversion.

      Add all required properties to mappingBuilder before passing it as argument. (The Build method is called by the constructor.)
